## quick start

### install pixi

[prefix.dev](https://prefix.dev/)

```
# windows 
iwr -useb https://pixi.sh/install.ps1 | iex
# Linux & macOS
curl -fsSL https://pixi.sh/install.sh | bash
```

### start the project

```shell
pixi run start
```

it will generate a file named `tv.m3u8` , open it with potplayer or VLC or any player you like
